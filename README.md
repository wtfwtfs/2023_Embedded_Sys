# 🚪 8조 : 임베디드 시스템 프로젝트
## 주제
- 간단한 도어락 프로토콜을 구현합니다.
- RFID를 이용합니다.
- LCD를 이용해 도어락 상태를 표시하고, LED를 이용해 전등을 표현합니다.
- 블루투스로 상호작용합니다.

## 기능
1. 블투로 불 켜고 끄기
2. 카드로 문 열고 닫기
3. 블투로 문 열고 닫기
4. 비번으로 문 열고 닫기
5. 잘못된 비번이나 카드를 대면 침입자 알람
6. 비번이나 등록된 카드로 침입자 알람 끄기
7. 비번 변경
8. 카드 재등록
9. 물리적으로 버저 소리 줄이고 키우기
